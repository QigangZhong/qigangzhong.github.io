## 安装

### docker方式安装

```bash
docker pull rabbitmq:3.7.16-management
docker run -d --hostname my-rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.7.16-management
# 访问http://{host}:15672，默认username/pwd为guest/guest
```
